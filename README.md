# denis
# Конвертер систем счисления

Веб-приложение для конвертации чисел между системами счисления.

## Функциональность
- Конвертация из десятичной системы в двоичную
- Конвертация из десятичной системы в восьмеричную  
- Конвертация из десятичной системы в шестнадцатеричную
- Валидация ввода данных
- Обработка ошибок

## Технологии
- HTML5
- CSS3 (Grid, Flexbox, анимации)
- JavaScript (ES6+)

## Запуск
Откройте файл `index.html` в браузере

## Особенности
- Полностью клиентское приложение
- Адаптивный дизайн
- Валидация ввода
- Визуальная обратная связь
